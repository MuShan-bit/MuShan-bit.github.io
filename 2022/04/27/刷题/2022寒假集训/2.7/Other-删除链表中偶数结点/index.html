Problem Description

初始为空链表，接下来有两种操作：

1、给你一个数 k 添加到链表的末尾

2、以链表的第一个元素下标为 1，将链表中偶数下标的元素删除，没有元素则跳过

Input

第一行一个整数 n，表示操作次数（1≤n≤3e5）

接下来 n 行，每行先一个整数 op（1≤op≤2），

如果 op 为 1，则为操作 1，后面再输入一个整数 k（k 在 int 范围内）

如果 op 为 2，则为操作 2

保证操作 2 的次数≤10

Output

对于操作 2，输出每次删除后的链表
