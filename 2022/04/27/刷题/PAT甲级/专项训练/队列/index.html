# 1056 Mice and Rice

# 题目大意

输入一个 N 和一个 K, N 表示老鼠个数，随后两行

第一行 N 个数表示从 0 到 N-1 编号的老鼠的体重

第二行 N 个数表示老鼠的比赛顺序，每次按顺序选取 K 个对应编号的老鼠比体重，体重最高的晋级，未晋级的老鼠的排名将被确定为当前老师可分的组数 + 1

例如样例

选择编号为 6 0 8 的老鼠比较，编号为 8 的老鼠最大，晋级，编号为 6 和 0 的老鼠的排名为 5

因为 11 / 3 = 3, 因为余 2 所以还要再分一组，就是 4 组，然后组数 + 1, 所以排名就是 5

# 解题思路

用两个队列模拟即可
