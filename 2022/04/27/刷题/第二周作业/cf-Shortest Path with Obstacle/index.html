# A. Shortest Path with Obstacle

题意

给 N，给 N 组数据，每组中包括两个 A、B 点以及一个 F 点的二维坐标，求 A 至 B 的最短距离，但是不能经过 F。

这个最短距离不能斜着走.

解题思路

如果两点横坐标纵坐标都不同，不需要考虑会碰到 F, 因为这个最短路径至少有两条。但是如果有横坐标或者纵坐标相同了，那么最短路是一条直线，只有一条，(此时要判断是否经过 F, 经过就 + 2, 不经过不变) 如果横纵坐标都一样那就是一个点输出 0
