# 题目大意

就是求杨辉三角中某一行的乘积和对 1e9 + 7 取余

# 题目考点

  1. 杨辉三角的相关知识 (需要知道杨辉三角与组合数的关系)
  2. 求阶乘，求组合数基础知识
  3. 因为这道题中运用到了除法的求余，所以用到了逆元的知识
  4. 线性求逆元算法

# 线性求逆元算法公式

公式: inv[i]=(M−M/i)∗inv[M%i]%Minv[i] = ( M - M\ /\ i ) * inv[M\ \%\ i]\ \%\ Minv[i]=(M−M / i)∗inv[M % i] % M​
